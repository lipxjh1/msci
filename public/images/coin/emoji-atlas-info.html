How to Create the Atlas

  1. Use an image editing software like Photoshop, GIMP, or Aseprite.
  2. Create a new image with dimensions 256√ó192 pixels, with transparency.
  3. Create a grid with 4 columns and 3 rows, each cell being 64√ó64 pixels.
  4. Place each emoji in its corresponding position as shown above.
  5. Save the file as emoji-atlas.png in the public/images/coin/ directory.
  6. Ensure the JSON file emoji-atlas.json is already in place with the correct frame definitions.

Alternative: Using Emoji Images

If you have individual emoji images, you can use tools like TexturePacker or a simple script to arrange them into a sprite sheet.

Testing

After creating the atlas, test the game to ensure all emojis appear correctly. If emojis appear incorrectly or don't show up, double-check the atlas image's dimensions and the placement of each emoji.
